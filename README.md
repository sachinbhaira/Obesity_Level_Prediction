# Obesity Level Prediction using Machine Learning

## 📌 Project Overview
This project predicts a person's obesity level using Machine Learning algorithms based on physical characteristics, eating habits, lifestyle, and transportation habits.

## 🎯 Objective
To classify a person into different obesity levels using supervised machine learning models.

## 📊 Dataset
The dataset contains 17 input features, including:
- Gender
- Age
- Height
- Weight
- Family history with overweight
- High-calorie food consumption
- Vegetable consumption
- Number of meals
- Water intake
- Physical activity
- Technology usage
- Smoking
- Alcohol consumption
- Transportation mode
- And other lifestyle-related attributes

Target Variable:
- Obesity Level

## 🛠️ Technologies Used
- Python
- Jupyter Notebook (Anaconda)
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🤖 Machine Learning Models
- Decision Tree Classifier
- Random Forest Classifier

## ⚙️ Data Preprocessing
- Loaded dataset from Excel
- Handled categorical values using Label Encoding
- Split dataset into 80% training and 20% testing

## 📈 Results

| Model | Accuracy |
|--------|----------|
| Decision Tree | 84.37% |
| Random Forest | 89.79% |

## ▶️ How to Run
1. Clone the repository.
2. Install the required libraries:
   pip install -r requirements.txt
3. Open the notebook in Jupyter Notebook.
4. Run all the cells.
